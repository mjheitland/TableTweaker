TableTweaker

(©2019 Michael Heitland)

TableTweaker is a mighty tool to manipulate any text that has a table structure.
Copy and paste your table into the input field (bottom left window), use a regex to filter which input lines shall be processed, describe the transformation in the pattern field (top window) and check the transformed text in the output field (bottom right window). The output can be copied into any other program for further processing.
The highlight: you can use any scripted C# method you like. This program comes with a code editor supporting syntax highlighting and Intellisense to help you coding! The code gets compiled on the fly.
In manual mode you have to press F5 to run the transformation, in auto mode the transformation happens automatically whenever the input gets changed.
Set the field qualifier to none if you want to keep quotation marks of the input, otherwise they will be eliminated and a pair of quotation marks will be replaced by single quotation marks.
If you have a 1:1 mapping of one input line to one output line, you might find the Result Grid helpful that shows input and output as HTML table. That makes it easy to scroll through input and output synchronously. You can toggle between one and two column layout.

Initial Example: Mailing everyone from your contact list that an accident has happened


Keywords

$dollar;: "$" (dollar sign)
$h0, $h1, ...: fields from the header row (i.e. first line of your input; index is 0-based)
$h-0, $h-1, ...: fields from the header row (right-most field has index -0, the next one -1 etc.)
$0, $1, ...: row fields (0-based)
$-0, $-1, ...: row fields (right-most field has index -0, the next one -1 etc.)
$header: first row
$row: current row
$rowNum: row number (0-based)
$rowNumOne: row number (1-based)
$numFields: number of fields per row
$numRows: total number of rows in the input field
$MyMethod("$0",$1,$row, 7, "some text", ...): call any C# method from your code window. If you use ")" within your args, pick another set of brackets instead of "(" and ")", e.g. [], <> or {}.

Section markers

$ONCE: apply the pattern only once
$EACH: apply the pattern for every single row of the input field
$EACH+: apply the pattern for every single row of the input field, but skip the first row (i.e. header line)

Prebuilt Functions

FormatDate: use a format string to describe how you want to have your dates displayed
IndexOf: searches for a pattern and returns its index within the string or -1 if not found
Left: returns the "n" left-most characters
Right: returns the "n" right-most characters
Replace: replace a substring with another substring
Substring: returns a substring
ToLower: converts a string to lower case
ToUpper: converts a string to upper case
Trim: removes leading and trailing characters
